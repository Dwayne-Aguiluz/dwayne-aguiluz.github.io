All about CSS

  • CSS is the language we use to style a Web page.
  • CSS Stands for Cascading Style Sheets
  • CSS separates content from design, making websites easier to maintain and more attractive.
  • Saves time: one CSS file can control multiple pages.
  • Improves accessibility and responsiveness.

Benefits of CSS

  • CSS saves a lot of work
  • Easy maintenance
  • Adding style to webpage
  • CSS allows you to position objects in three dimensions.
  • CSS allows you to create custom tags to achieve specialized objectives
  • CSS gives you control when placing objects on the page exactly where you want them.
  • Download faster because your browser will download only one file once

Layers Of A Webpage



there are 3 types

  1. Presentation Layer
  2. Behavior Layer
  3. Content Layer

Content layer

It is where you define and layout the text, images, animation, sound,video and everything you want to put in a web page with the use of tags.

is the underlying HTML code of that page. It is a solid foundation of HTML creates a platform upon which a website can be created

  • Text that makes sense without a visual representation.
  • Alternative text for every image, sound piece or video that is content
  • Explanations of Acronyms and Abbreviations
  • Text that is fit for the web (KISS, structured intoheaders, paragraphs and lists)

Presentation Layer

The presentation layer is where you can define how people see your web page in a browser

  • ensure that text can be zoomed without making the site unusable
  • ensure that images and foreground and background hav enough contrast and are unambiguous to the colorblind
  • ensure that the interactive elements of the site are easy to find

Behavior Layer

The behavior layer is where you have a real-time useinteraction with the web page. The interaction may be simple validation of input or filling out a form or even abig as web-based programs.

  • ensure that all the functionality is available to the user regardless of input device
  • make the user experience as easy as possible by cuttindown on options until they are necessary
